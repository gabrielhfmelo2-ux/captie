## 📌 Sobre o Projeto
O **Captie** é um dashboard financeiro interativo desenvolvido em Python. A aplicação visa trazer clareza para investidores, simulando o efeito dos juros compostos com descontos reais de Imposto de Renda e a análise crítica da corrosão do poder de compra pela inflação histórica brasileira.

Diferente de calculadoras comuns, o Captie foca na **Experiência do Usuário (UX)** com um design *Glassmorphism* moderno, integrando dados econômicos reais (IPCA) e variações de preços de mercado desde 1995.

## ✨ Funcionalidades

* **📊 A Jornada:** Projeção de patrimônio comparando "Dinheiro Parado" vs "Poupança" vs "Investimento Eficiente", com rastreio de marcos (R$ 100k, 500k e 1M).
* **🏖️ Viver de Renda:** Cálculo baseado na metodologia F.I.R.E. (*Financial Independence, Retire Early*), mostrando quanto o patrimônio gera de salário mensal isento.
* **🛒 Poder de Compra Real:** Uma "Máquina do Tempo" que utiliza o histórico do IPCA para converter o valor acumulado no futuro para o equivalente em anos passados, mostrando o impacto real da inflação no consumo.

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Python 3.10+
* **Interface:** Streamlit (Customizado com CSS Injection)
* **Processamento de Dados:** Pandas
* **Visualização de Dados:** Plotly (Graph Objects)
